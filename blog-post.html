Portfolio Overview

What This Portfolio Demonstrates

This SolidWorks CAD portfolio covers 11 projects spanning aerospace assembly, power transmission, motion mechanisms, product enclosure design, and CAD-to-hardware fabrication. Each project reflects disciplined assembly methodology — interface-first mating, functional grouping, documentation-ready outputs, and validation checks.

Project 01

Aircraft Landing Gear Assembly

SolidWorks Aerospace Assembly Technical Drawing

A twin-wheel landing gear concept built to demonstrate aerospace-style assembly discipline. The model features a primary structural leg, support linkage, and wheel-axle integration with clear load-path visibility. Emphasis was placed on packaging clarity and dimensioned drawing communication.

Key Engineering Decisions
  • Interface-first approach — wheel/axle axis treated as controlling geometry
  • Structural separation between primary leg and support linkage for load-path readability
  • Section-style inspection view to expose internal envelope and clearances
  • Dimension-led documentation for design review
Project 02

Manual Gearbox Assembly

SolidWorks Power Transmission Multi-Stage Gear Train

A full multi-stage gear train assembled along parallel shafts, with an external shift lever and selector linkage. This was an assembly-only contribution — the challenge was bringing a large part count into a coherent, rebuildable system without mate conflicts.

Assembly Strategy
  • Axis-first mating — primary shaft axes used as master references
  • Stack-up logic for correct gear spacing along each shaft
  • Functional grouping to keep gear train and shift mechanism visually distinct
  • Configuration control for internal and external presentation views
Project 03

Mastering Pump Gearbox Assembly

SolidWorks Twin-Gear Housing Technical Drawing

A compact twin-gear pump gearbox with a bolted split housing. The key deliverable was a full A3 drawing sheet with section views (Section B-B and F-F) communicating the internal gear train, bearing placement, and cavity geometry — making the design reviewable and serviceable.

Design Decisions
  • Split housing with bolted cover for practical serviceability
  • Section-driven documentation to expose internal arrangement
  • Interface-first assembly with shafts and bearings as controlling geometry
  • Fastener layout around functional envelope for clamping and tool access
Project 04

Transmission Dog Clutch Assembly

SolidWorks Positive Engagement Exploded Views

A dog clutch sub-assembly representing a positive mechanical engagement system. Unlike friction clutches, dog clutches transfer torque through mechanical interlock — so axial stack-up, engagement geometry, and retention logic must be precisely modelled. Both assembled and exploded views were produced to communicate build order.

Validation Checks
  • Coaxial alignment check across all rotating elements
  • Interface and clearance check — no overlap in assembled config
  • Exploded sequence verified against a realistic build order
  • Stack-up and retention logic with spacers, washers, and retainers
Project 05

Pulley-Shaft Support Assembly

SolidWorks Interface Definition Exploded View

A compact assembly demonstrating clean interface definition between a shaft, key, bush, and pulley. The primary output is exploded-view communication — showing keyed torque transfer, bushing/shaft relationships, and a manufacturable mounting geometry.

What This Demonstrates
  • Exploded-view communication of build order and functional interfaces
  • Keyed connection and bushing/shaft relationships in the model
  • Axis alignment and part orientation preserved across configurations
  • Manufacturable support geometry and mounting interfaces
Project 06

Linear Actuator Stage Assembly

SolidWorks Lead Screw Drive Motion System

A lead screw linear stage integrating a motor mount, coupling region, guided carriage, and rail. The goal was clean motion-system integration — converting motor rotation into controlled linear travel while maintaining alignment between screw axis, guide axis, and carriage travel direction.

Design Requirements Met
  • Motor rotation → controlled linear travel via lead screw
  • Carriage constrained for straight-axis translation with minimal rotation
  • Drive, guide, support, and carriage structured as functional groups
  • Axis alignment maintained across all moving interfaces
Project 07

Linkage-Driven Linear Actuator

SolidWorks Kinematics Rotary-to-Linear

A compact mechanism that converts rotary input into guided linear output using a pivoted linkage cluster and a sliding output block. The challenge was maintaining exactly one degree of freedom so the mechanism behaves predictably without over-constraining.

Mechanism Requirements
  • Single DOF — rotational input → stable linear displacement
  • No over-constraints in the assembly mates
  • Slider path defined and verified for minimal unwanted rotation
  • Compact, mechanically interpretable layout
Project 08

Raspberry Pi Holder & Enclosure

SolidWorks Reverse Engineering Product Design

A reverse-engineered Raspberry Pi enclosure with a split top/bottom housing, internal PCB locating features, accurate port cut-outs, and ventilation slots. The project demonstrates the ability to capture real board geometry and design a manufacturable case around it.

Design Criteria
  • Board and connector envelope captured for zero-interference fit
  • Port openings accurately positioned for USB, HDMI, SD, and power
  • PCB secured using internal locating features — no fasteners into board
  • Ventilation features balanced with structural stiffness requirements
Project 09

Spring Latch Bolt — CAD to Real Product

SolidWorks Product Design Physical Build

This is the portfolio's only CAD-to-hardware project. A spring latch bolt was modelled in SolidWorks, then fabricated as a real product. The design uses a sliding bolt, compression spring, and guide structure — the final fabricated piece closely matches the CAD model, validating the design decisions made upstream.

Why This Matters
  • Proves CAD-to-manufacture workflow, not just modelling ability
  • Spring return action verified on the physical product
  • Standard hardware used wherever possible to reduce fabrication complexity
  • Mounting features and tool access preserved through design iterations
Project 10

Geneva Mechanism

SolidWorks Intermittent Motion Indexing Drive

A Geneva mechanism that converts continuous driver rotation into step-by-step indexed rotation, with a clean dwell period between each step. This project is about geometry precision — the engagement and disengagement of the drive pin with the Geneva slots must be timed correctly to prevent interference.

Design Requirements
  • Continuous input → discrete indexed rotation of Geneva wheel
  • Clear dwell period between each indexed step
  • Pin-slot geometry checked for interference-free engagement
  • Single DOF driven by one input rotation
Project 11

Four-Bar Linkage Mechanism

SolidWorks Planar Kinematics Motion Study

A planar four-bar linkage built to demonstrate fundamental kinematic behaviour. The focus is on correct degree-of-freedom definition through joint types, and observing how link-length ratios control coupler trajectory. Motion frames were captured from the SolidWorks motion study.

What Was Validated
  • True planar mechanism — revolute joints at all link connections
  • Smooth motion with no unintended DOF or mate conflicts
  • Link-length relationships control coupler path as expected
  • Motion study captures behaviour across full crank rotation
